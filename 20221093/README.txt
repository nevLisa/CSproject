테스트 파일 입니다.

데이터 용량이 커서 경량 데이터로 테스트 해서 만든 코드입니다.
원본 데이터를 사용할때 데이터 폴더를 수정하거나 경로 코드를 살짝 수정하면 될 것 같습니다.
돌아가는것 까지만 테스트 해 봤습니다.

사용한 데이터 셋
https://aihub.or.kr/aihubdata/data/view.do?srchOptnCnd=OPTNCND001&currMenu=115&topMenu=100&searchKeyword=%EB%8F%99%EC%9D%98%EB%B3%B4%EA%B0%90&aihubDataSe=data&dataSetSn=151

파일 초간단 설명

data_manager.py = 이미지별 데이터 프레임을 만들어 주는 코드
dataset.py = 이미지들을 학습에 좋게 조절하고 데이터 프레임과 합치는 코드
utilities.py = 기타 자잘한 코드들 (현재는 없음)
training.py = 모델 학습할때 쓰는 코드
